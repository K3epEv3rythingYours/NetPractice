### Variable Length Subnet Masking

VLSM allows **different subnet sizes** within the same network.

**Three-Way Handshake:**
```

  |-------- SYN ----------->|   "Want to connect"
  |<----- SYN-ACK ---------|   "OK, acknowledged"
```

### UDP: Fast but Unreliable

**No handshake. Fire and forget.**

### Comparison

| Feature | TCP | UDP |
|---------|-----|-----|
| Connection | 3-way handshake | None |
| Reliability | Guaranteed | Best-effort |
| Speed | Slower | Faster |
| Use cases | Web, email, files | Gaming, VoIP, DNS |
